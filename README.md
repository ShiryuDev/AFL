### Setup

<details>
  <summary>Maven</summary>
  
  ```maven
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>

    <dependencies>
        <dependency>
            <groupId>com.github.ShiryuDev.AFL</groupId>
            <artifactId>bukkit</artifactId>
            <version>1.2</version>
        </dependency>
    </dependencies>
     
 ```
</details>
